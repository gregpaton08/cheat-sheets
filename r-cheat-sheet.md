# R Cheat Sheet
###### Verry much a work in progress

Get help
```
?somecommand
```

Set the width of the terminal.
```
options("width"=200)
```

Create a data vector.
```
c(1,2,3)  # Creates vector [ 1, 2, 3 ]
c(1:5)    # Creates vector [ 1, 2, 3, 4, 5 ]
```

Plot some values.
```
x=c(0:100)
y=c(100:200)
plot(x,y)
```

Get the dimensions of a data set.
```
dim(data)
```
